# Print Instructions · CX Leaders Advance 2026

## canvas.html · Session Canvas (18" × 24" Landscape)

**Paper size:** 18" × 24" landscape (609.6 mm × 457.2 mm) - Staples standard large format  
**Stock:** Matte (not gloss - participants need to write on it)  
**Quantity:** 1 per table + 1 spare = 6 copies minimum for 5 tables  
**Colour:** Full colour, backgrounds must print  

**Reusable option (future workshops):** Laminated + dry-erase markers (1 set per table), or 200 gsm uncoated if participants keep it as a takeaway. CXLA 2026 canvas is event-specific - no lamination.

**PDF export:** Custom paper size 24 × 18 inches in Chrome print dialog (width × height, landscape). To get inches in the dialog: System Preferences > General > Language & Region > Measurement system > Imperial, then restart Chrome.

**At Staples Canada online:** Upload PDF, select **18" × 24"** large format poster, matte finish, full colour. PDF is sized to match exactly - select "print at actual size."

## casecards.html · Case Cards (6" × 9" Portrait)

**Paper size:** 6" × 9" portrait - Staples standard size, close match to card design (148 mm × 210 mm). Small white strip at bottom is acceptable.  
**Stock:** 100 lb white card stock, matte (uncoated) - participants write on the notes lines. Staples offers matte or gloss; choose matte.  
**Quantity:** 1 set of 5 cards per table = 25 cards minimum for 5 tables. Print extra sets if you expect groups to keep them.  
**Colour:** Full colour, backgrounds must print  

**At Staples Canada online:** Upload PDF, select **6" × 9"** card stock, 100 lb, matte, full colour. Cards are one per page in the PDF.

**Cutting note:** Cards print one per page at 6"×9" - no cutting needed. If Staples defaults to 8.5"×11", cut to 6"×9" with a guillotine cutter.

## slides.html · Onscreen Presentation PDF (16:9)

**Intended use:** Screen display only. Send to Gabe as a PDF file. Do not print.  
**Page size (embedded):** 254 mm × 143 mm (16:9 widescreen)

### Export steps (Chrome on Mac)

1. Open `slides.html` in Chrome
2. Press `Cmd + P`
3. Click **More settings**
4. Check **Background graphics** - this is required for all colour backgrounds and stats to appear
5. Set **Paper size** to match 254 mm × 143 mm (or use the embedded page size - Chrome will respect `@page` dimensions when saving as PDF)
6. Set **Margins** to None
7. Click **Save as PDF**

---

## Mac System Dialog - Background Fix

If Chrome's built-in print dialog drops backgrounds (common on macOS), switch to the system dialog:

1. In Chrome's print dialog, click **More settings** → scroll to the bottom → **Open in Preview** or choose **Print Using System Dialog**
2. In the macOS system dialog, go to **Safari/Chrome** dropdown → **Layout** → check that colour output is selected
3. The `print-color-adjust: exact` rule is already applied to both `canvas.html` and `casecards.html` - backgrounds should render correctly via the system dialog
